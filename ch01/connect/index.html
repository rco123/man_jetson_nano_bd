보드 IP 연결

USB 보드 연결

USB 유선 케이블 연결을 사용하면 (노트북) PC에서 Jetson nano Board의 login 터미널에 네트워크를 통하여 접속할 수 있습니다.

가상 이더넷 어댑터 기능을 통해 Jetson Board는 PC와 Point-to-Point 네트워크를 연결할 수 있습니다. Jetson Board의 Default 고정 IP는 ”192.168.55.1“ 입니다.

Jetson Board외 연결되는 PC의 IP를 192.168.55.100로 설정하고 가상 로컬 이더넷 네트워크를 형성합니다. 따라서 Jetson Nanod의 계정 (사용 이름:nano 및 암호 사용:1111)을 생성한 경우, 네트워크 Jetson 보드의 접속 IP를 192.168.55.1을 지정하여 PC에서 SSH를 이용하여 Jetson Board로 로그인 할 수 있습니다.

나노 보드 USB IP 접속

  • login ID : jetson
  • PASSWD : 1111
  • IP ADDRESS : 192.168.55.1

Wifi Address 설정

USB를 컴퓨터와 연결 후, 192.168.55.1 을 통하여 웹브라우즈로 iblock을 브라우즈 실행

iBlock 연결

주소 입력후 연결

웹브라우즈를 열고, 주소를 입력하여 젯슨보드에 접속한다.

iBlock 의 툴박스에서

코드 => 인터넷 => 와이파이 설정 블록을 작업창으로 가지고 와서 실행을 한다.

사용중인 무선 공유기의 SSID , 비밀번호를 입력하여 블록 프로그램을 수행한다.

프로그램 실행 및 결과 확인

실행하기 버턴을 클릭한다. 블록으로 구성된 프로그램이 실행되고, 보드의 무선 IP를 설정한다.

무선이 연결되면, USB 케이블을 제거하고, 새로 연결된 무선랜 IP를 이용하여 보드에 접속한다.
