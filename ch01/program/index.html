연결 프로그램

일반적으로 많이 사용하는 터미널 프로그램 2가지를 소개한다. 프로그램 작성을 위해서는 터미널 프로그램이 필요하다.

MobaXterm

vi를 이용하여 프로그램 작성 및 보드 조작을 수행하는 사용자에게 편리한 프로그램이다.

vs code

윈도우 환경에서 프로그램 작성에 익숙한 사용자에게 적합한 프로그램이다.
