파이토치 프로그램

파이토치 프로그램을 작성하고 실행하기 위한 기초적인 과정에 대해서 설명한다.

디렉토리 생성

먼저 1장에서 설명한 터미널 프로그램을 이용하여 보드에 접속한다. 접속후 다음 실행명령어를 이용하여 프로그램을 작성한다.

    cd ~           # 홈디렉토로 이동
    mkdir code     # 디렉토리가 없는 경우 생성
    cd code        # 생성된 디렉토리로 이동
    vi run.py      # 파이썬 프로그램 작성을 위한 에디터 프로그램(VI) 실행

프로그램 작성

아래의 프로그램을 작성하여 동작 테스트를 수행한다.

import torch

def test_pytorch():
    # PyTorch 버전 출력
    print("PyTorch Version:", torch.__version__)

    # GPU(CUDA) 사용 가능 여부 확인
    if torch.cuda.is_available():
        device = torch.device("cuda")
        print("CUDA is available. Using GPU:", torch.cuda.get_device_name(0))
    else:
        device = torch.device("cpu")
        print("CUDA is not available. Using CPU.")

    # 간단한 텐서 연산 테스트
    print("\nTensor Operations Test:")
    x = torch.tensor([1.0, 2.0, 3.0], device=device)
    y = torch.tensor([4.0, 5.0, 6.0], device=device)
    z = x + y
    print("x:", x)
    print("y:", y)
    print("z = x + y:", z)

    # 텐서 행렬 곱셈 테스트
    print("\nMatrix Multiplication Test:")
    a = torch.randn(3, 3, device=device)
    b = torch.randn(3, 3, device=device)
    c = torch.matmul(a, b)
    print("Matrix A:\n", a)
    print("Matrix B:\n", b)
    print("Matrix C (A @ B):\n", c)

    # GPU 속도 테스트
    if torch.cuda.is_available():
        print("\nRunning a simple computation on GPU...")
        gpu_tensor = torch.randn(1000, 1000, device="cuda")
        result = torch.matmul(gpu_tensor, gpu_tensor)
        print("Computation successful on GPU.")
    else:
        print("\nSkipping GPU computation as CUDA is not available.")

if __name__ == "__main__":
    test_pytorch()

코드 설명

PyTorch 버전 확인:

torch.__version__을 출력하여 PyTorch의 설치 버전을 확인합니다.

CUDA 사용 가능 여부 확인:

torch.cuda.is_available()로 GPU 사용 가능 여부를 체크합니다. GPU가 사용 가능하면 torch.cuda.get_device_name(0)으로 GPU 이름을 출력합니다.

기본 텐서 연산:

torch.tensor를 사용해 텐서를 생성하고, 기본적인 텐서 연산(덧셈 등)을 수행합니다.

행렬 연산 테스트:

torch.randn으로 난수 행렬을 생성한 후, torch.matmul로 행렬 곱셈을 수행합니다.

GPU 연산 테스트:

GPU가 사용 가능할 경우, 큰 텐서를 생성하고 행렬 곱셈을 수행하여 GPU 연산 테스트를 합니다.

작성된 프로그램 실행

cd ~/code
python run.py
