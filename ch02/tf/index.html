텐스플로우 프로그램

tensorflow 프로그램을 작성하고 실행하기 위한 기초적인 과정에 대해서 설명한다.

디렉토리 생성

먼저 1장에서 설명한 터미널 프로그램을 이용하여 보드에 접속한다. 접속후 다음 실행명령어를 이용하여 프로그램을 작성한다.

    cd ~           # 홈디렉토로 이동
    mkdir code     # 디렉토리가 없는 경우 생성
    cd code        # 생성된 디렉토리로 이동
    vi run.py      # 파이썬 프로그램 작성을 위한 에디터 프로그램(VI) 실행

프로그램 작성

아래의 프로그램을 작성하여 동작 테스트를 수행한다.

import tensorflow as tf

def test_tensorflow():
    # TensorFlow 버전 출력
    print("TensorFlow Version:", tf.__version__)

    # GPU 사용 가능 여부 확인
    print("GPU Available:", tf.config.list_physical_devices('GPU'))
    if tf.test.is_gpu_available():
        print("GPU is available.")
        print("GPU Device Name:", tf.test.gpu_device_name())
    else:
        print("No GPU detected. Running on CPU.")

    # 간단한 텐서 생성 및 연산 테스트
    print("\nTensor Operations Test:")
    a = tf.constant([[1, 2], [3, 4]], dtype=tf.float32)
    b = tf.constant([[5, 6], [7, 8]], dtype=tf.float32)
    result = tf.add(a, b)  # 텐서 덧셈
    print("Tensor A:\n", a.numpy())
    print("Tensor B:\n", b.numpy())
    print("A + B:\n", result.numpy())

    # 행렬 곱셈 테스트
    print("\nMatrix Multiplication Test:")
    result = tf.matmul(a, b)
    print("A @ B:\n", result.numpy())

    # GPU 연산 테스트
    if tf.test.is_gpu_available():
        print("\nRunning a simple computation on GPU...")
        with tf.device('/GPU:0'):
            large_tensor = tf.random.uniform([1000, 1000])
            result = tf.matmul(large_tensor, large_tensor)
            print("GPU computation successful.")
    else:
        print("\nSkipping GPU computation as no GPU is available.")

if __name__ == "__main__":
    test_tensorflow()

코드 설명

  • TensorFlow 버전 확인:

    • tf.__version__을 사용해 현재 설치된 TensorFlow의 버전을 출력합니다.

  • GPU 사용 가능 여부:

    • tf.config.list_physical_devices('GPU')로 GPU가 사용 가능한지 확인합니다.

  • 텐서 생성 및 연산:

    • tf.constant를 사용하여 정수 배열 텐서를 생성합니다. tf.add를 사용해 두 텐서를 더합니다.

  • 결과 출력:

    • .numpy()를 호출해 텐서 데이터를 numpy 배열로 변환하고 출력합니다.

작성된 프로그램 실행

cd ~/code
python run.py
