카메라

NVIDIA Jetson 보드(예: Jetson Nano, Xavier NX 등)에 IMX219 카메라 모듈을 연결하고 사용하는 방법은 다음과 같습니다. IMX219는 Raspberry Pi 카메라 V2와 동일한 센서를 사용하며, Jetson 보드의 CSI(Camera Serial Interface) 커넥터와 호환됩니다.

필요한 하드웨어

  • NVIDIA Jetson 보드 (예: Jetson Nano, Xavier NX 등).
  • IMX219 카메라 모듈.
  • CSI 케이블 (보드와 카메라를 연결하는 리본 케이블).

카메라를 그림과 같이 연결한 경우, 카메라로 부터 영상을 입력되는 영상을 확인하는 과정을 다음과 같이 수행한다.

Note

192.168.0.27 주소는 현재 젯슨 보드에 할당된 IP 주소에 따라 변경하여 사용한다.

툴박스 => 코드 => 로보 => 04.영상출력 블록을 선택한다.

생성된 블록과 블록으로 생성된 파이썬 코드를 실행하기 버턴을 클릭하여 실행한다.

파이썬 코드를 실행하고, 영상보기 버턴을 클릭하면, 카메라 영상을 확인할 수 있다.
