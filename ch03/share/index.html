파일 공유

PC 에서 Jetson Nano 보드 쪽으로 파일을 보내거나 수신하기 위하여, 젯슨 보드에서 공유디렉토리를 제공한다. 이를 통하여 Jetson Board 에 있는 파일을 내용을 확인할 수 있다.

파일탐색기(Window Explore)를 실행한 후, 주소창에 ₩₩ 192.168.0.27 주소를 입력 그리고 엔터키를 클릭하는 경우, 파일탐색기는 젯슨 보드의 디렉토리의 파일을 검색할 수 있도록 한다.

접속된 디렉토리는 홈디렉토리 아래의 share 디렉토리를 공유한다.

/home/jetson/share  # 지정된 공유디렉토리

Note

192.168.0.27 주소는 현재 젯슨 보드에 할당된 IP 주소에 따라 변경하여 사용한다.

Tip

처음 접속 하는 경우, ID 와 passwd 를 물어보는 경우,

  • ID : jetson
  • password : 1111

를 입력하고 접속을 한다.

접속이 정상적으로 수행되면 젯슨 보드에 있는 파일을 PC에서 읽고 쓰기를 할 수 있다.
