IP 주소표시

젯슨 보드에 OLED 0.96인치 디스플레이를 통해서 현재 설정되어 있는 IP 주소를 확인

젯슨 보드는 부팅 후, 현제 설정된 IP 주소를 표시한다.

USB로 케이블이 연결 된 경우는 192.168.55.1 으로 기본 주소를 표시하여 주며, 무선으로 IP 주소를 설정된 경우는 해당 192.168.XX.XX 로 표시된다.

무선의 경우 한번 설정이 되면, 재 부팅하여도, 해당 주소를 유지하고 있으며, 한번 설정후에는 반복해서 설정과정이 필요없게 된다.

부팅후 젯슨 보드에 설정된 IP 주소를 확인하고 터미널 또는 웹브라우즈를 통한 iBlock 프로그램을 수행할 수 있다.
